Connect your data store

This app uses JSONBin.io as a free shared database so your whole team sees the same data. Follow these steps to set it up:

1
Go to jsonbin.io and create a free account.
2
After logging in, click API Keys in the sidebar. Copy your Secret Key (starts with $2b$...).
3
Click Bins → Create Bin. Paste {"stories":[]} as the content and save. Copy the Bin ID from the URL.
4
Paste both below and click Connect. Share the same API key + Bin ID with your team so everyone connects to the same data.
